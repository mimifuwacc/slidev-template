# Welcome to [Slidev](https://github.com/slidevjs/slidev)!

This project uses [Vite+](https://viteplus.dev/) (`vp`) as its toolchain.

To start the slide show:

- `vp install`
- `vp run dev`
- visit <http://localhost:3030>

Edit [src/slides.md](./src/slides.md) to see the changes.

## Other commands

- `vp run build` — build the static site to `../dist`
- `vp run export:pdf` — export slides to `output/slides.pdf`
- `vp run export:pptx` — export slides to `output/slides.pptx`
- `vp check` / `vp test` — format, lint, type check and test

Learn more about Slidev at the [documentation](https://sli.dev/).
